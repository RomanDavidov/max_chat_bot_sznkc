🩺 Для Врачей(Стационар КБ122)

Инструкции для врачей стационара КБ122

Ведение истории болезни пациента

####


ШАГ 1: Войдите в систему АРМ «Врач стационара»: введите свой пароль и нажмите кнопку «ОК»;



ШАГ 2: Если у врача несколько отделений: выберите отделение в соответствии с тем, где сейчас работаете;
ШАГ 3: Подтвердите вход: нажмите кнопку «Да»;

ПРИМЕЧАНИЕ 1: Обратите внимание на доступность отделений: врачам стационара будут доступны отделения по подразделениям (СПб, ВММЦ);
ПРИМЕЧАНИЕ 2: Учитывайте ограничения прав доступа: вести историю болезни (ИБ) возможно только в тех отделениях, за которыми врач закреплен, в иных отделениях доступен только просмотр.




ШАГ 4: Перейдите к истории лечения: откройте вкладку «История», выберите нужного пациента в списке справа и двойным щелчком левой кнопки мыши откройте текущую госпитализацию;



ШАГ 5: Начните ведение истории болезни: перейдите исключительно во вкладку «Дневники», так как историю болезни ведем только в «Дневниках»;
ШАГ 6, 7: Добавьте новый протокол: нажмите кнопку «+» и выберите нужный профиль из списка (ОБЩИЙ ПРОФИЛЬ);
ШАГ 8: Выберите необходимый протокол: найдите нужный протокол в предложенном списке;
ШАГ 9: Подтвердите выбор протокола: нажмите кнопку «Да»;

ВАЖНО: Контролируйте обязательный состав документов: на истории болезни (ИБ) ОБЯЗАТЕЛЬНО должны присутствовать
«Первичный осмотр»
«Выписной эпикриз».





ШАГ 10: Заполните медицинские данные: внесите всю необходимую информацию в пустые поля выбранного протокола;

ШАГ 11: Сохраните внесенные изменения: нажмите кнопку «Сохранить» для записи данных в систему;

ШАГ 12: Распечатайте медицинский документ, если требуется: нажмите кнопку «Печать» (справа- документы, которые подписываем ЭЦП (различные эпикризы) ;вывод Левая кнопка печати «Печать записей»: для вывода на печать иных документов (например: дневники и тд.);

ШАГ 13: Подпишите документ электронной цифровой подписью: выполните подписание ЭЦП (на данный момент подписываем только эпикризы: выписной, предоперационный, этапный, переводной и протокол оперативного вмешательства);

Перед подписанием документов проверяем вкладку «Диагнозы»





ШАГ 14: Переходим на вкладку «Диагнозы» и выбираем нужные коды по МКБ для «ПОСТУПЛЕНИЯ» И «ВЫБЫТИЯ».
Для пациентов с диагнозами C-необходимо добавлять в «Дневники» Onco_Лист.

Добавление диагностических назначений пациенту





ШАГ 1: Выберите вкладку: — «Диагностика»;

ШАГ 2: Создайте направление: нажмите кнопку «Добавить»;

ШАГ 3: Выберите из списка необходимый тип исследования, далее вид исследования.

ШАГ 4: Подтверждаем по кнопке «ДА»

ШАГ 5: По кнопке «Печать» печатаем направление;





Далее заполняем нужные поля в направлении, ( проверяем шифр в направлении и укажите цель исследования; ) и нажимаем кнопку «Сохранить».

Добавление лабораторных назначений пациенту




ШАГ 1: Выберите вкладку: — «Лаборатория»; - Создайте направление: нажмите кнопку «Добавить»

ШАГ 2: Выберите из списка необходимый тип материала, далее исследования- справа отобразиться заказ;
Лабораторные исследования можно выбрать по материалу исследования\группе\шаблону;

ШАГ 3: Подтверждаем по кнопке «ДА»

ШАГ 4: Направление появляется в списке лабораторных исследований, после чего при необходимости его можно изменить \ удалить, если всё добавлено, верно, то «Подтверждаем» исследование по кнопке

ШАГ 5: По кнопке «Печать» печатаем направление;

Добавление операции





ШАГ 1: В разделе текущая госпитализация в левом меню –нажимаем на вкладку «ОПЕРАЦИЯ»
ШАГ 2: Нажмите кнопку «+Добавить» для создания новой записи;
ШАГ 3: Заполните карточку операции: внесите данные в пустые поля карточки (порядок, вид операции, вид анестезии и т.д.);
ШАГ 4: Чуть ниже в блоке «Операции» нажмите «+Добавить»,
ШАГ 5: Выберите из предложенного списка нужную операцию и подтвердите выбор по кнопке «ДА»;





ШАГ 6: Сформируйте операционную бригаду: добавьте врачей и их роли по кнопке «+Добавить»

ШАГ 7: Доступен точечный сбор бригады по одному сотруднику или выбор сразу всей готовой бригады - подтвердите выбор по кнопке «ДА»;






ШАГ 8: Перейдите к созданию протокола операции: откройте вкладку «Протоколы» и нажмите кнопку «+Добавить» - далее «ОБЩИЙ ПРОФИЛЬ», после чего система автоматически добавит протокол оперативного вмешательства;
ШАГ 9: Заполните и подпишите протокол: внесите нужные медицинские данные в поля документа,
ШАГ 10: Нажмите кнопку «Подписать » и подпишите его ЭЦП, после чего по кнопке «Печать» распечатайте заполненные данные.
Примечание: заполнение строк с лучевой нагрузкой обязательны!

Закрытие госпитализации




ШАГ 1: Инициируйте закрытие госпитализации: найдите пациента в списке отделения на вкладке «Истории», выделите строку с его данными одним нажатием левой кнопки мыши и нажмите кнопку выписки;

ШАГ 2: Нажмите кнопку "ВЫПИСКА" (по этой же кнопке доступно действие перевод на отделение)

**ШАГ 3: Заполните обязательные поля выбытия: в открывшемся окне внесите данные в поля «Дата выбытия», «Тип выбытия», «Лечащий врач» и «Исход» - подтвердите действия по кнопке «ДА»**;

Закрытие истории болезни

Для того, чтобы пациент был полностью выписан из стационара, необходимо закрыть госпитализацию.



В открытой ИБ - необходимо перейти во вкладку «Диагнозы» - проверяем данные (ДИАГНОЗ ПОСТУПЛЕНИЯ И ВЫБЫТИЯ) – нажимаем кнопку «Закрыть» -подтверждаем по кнопке «ДА»;

После появиться надпись: История закрыта лечащим врачом.

Оформление консультаций пациентам с других отделений




ШАГ 1: Открываем вкладку «Журнал» - «Консультации внутр.» будут отражаться пациенты, которых направили к Вам на консультацию (Направление создает лечащий врач);

ШАГ 2: В нижней части экрана в разделе протоколы нажимаем «+» добавить протокол для оформления консультации

Общие сведения по работе в АРМ "Врач стационара"





Во вкладке «ИСТРОРИЯ» в центральной части экрана доступны кнопки
 «Карточка пациента» – редактирование личных данных;
 «Сменить» – для смены койки, профиля или лечащего врача;
 «Выписка» – Закрытие госпитализации, перевод на отделение, перевод в другое ЛПУ;




В верхней части окна - горизонтально расположены вкладки:
— «Текущая госпитализация» – ведение медицинской информации;
— «Карточка пациента» – редактирование личных данных;
— «Шифры» – редактирование данных о плательщиках и полисах;
— «Диагнозы» - ведение диагнозов по текущей госпитализации (поступления и выписки);
— «Услуги» – список услуг, входящих в текущую госпитализацию.
В верхней части окна - вертикально расположены вкладки:
— «Все события» – включает в обобщенном виде всю информацию, представленную в других блоках;
— «Дневники» – данные по наблюдениям , ведение ИБ;
— «Лист назначении» – представляет собой таблицу, столбцы которой являются временными интервалами длиной в астрономические сутки. Каждый астрономические сутки могут быть разделены на дополнительные интервалы. Наиболее часто используемые интервалы - это «Ночь», «Утро», «День», «Вечер».

— «Диагностика» – данные по диагностическим исследованиям - направления и результаты;
— «Лаборатория» – данные по лабораторным исследованиям - направления и результаты;
— «Операции» - данные по операциям;
— «Просмотр ИБ.» - просмотр всех данных по текущей ИБ.




— Просмотреть ИБ любого пациента можно нажатием правой кнопкой мыши –выбрать из списка «История лечения»

8. Новый - верный протокол оперативного вмешательства;

Алгоритм работы регистратора в АРМ "Приемное отделение"

*Регистрация пациента в приемном отделении*

ШАГ 1: Инициируйте регистрацию: Выберите способ оформления пациента (Вариант 1 или Вариант 2)



ВАРИАНТ 1: во вкладке «Приемное отделение» (она откроется автоматически) нажмите кнопку «+Принять».
• Проверьте дату (автоматически проставляется текущая) и правильность выбора приемного отделения;
• Нажать кнопку «Принять».



ВАРИАНТ 2: (Из листа ожидания) : Перейдите во вкладку «Очередь на госпитализацию».
• Проверьте установленный период дат и правильность выбора приемного отделения;
• Выделите нужного пациента из списка одним нажатием левой кнопки мыши и нажмите кнопку «Принять».
• Можно воспользоваться поиском нажав кнопку «Поиск в листе ожидания».



ШАГ 2: Найдите пациента: в открывшемся окне заполните поля поиска, нажмите кнопку «Найти» (или клавишу Enter) и выделите найденного пациента мышкой;
ШАГ 3: Продолжите оформление: нажмите кнопку «Дальше».

Если такого пациента нет в базе, то необходимо создать его карту.



ШАГ 4: По кнопке "Дальше" перейдите на вкладку «Личные данные». Введите новые данные или проверьте корректность существующих, если пациент уже был зарегистрирован ранее.
• Если пациент поступает по программе ОМС, выполните сверку полиса ОМС..
Обязательно проверьте заполнение следующих полей:
• Пол пациента
• СНИЛС
• Документ, удостоверяющий личность
• Адрес регистрации




ШАГ 5: Проверьте шифры и полисы: Перейдите на вкладку «Шифры и полисы». Проверьте корректность заполнения данных или введите новые шифры/полисы при их изменении.



ШАГ 6: По кнопке "Дальше" перейдите на вкладку «Родственники», затем по этой же кнопке в раздел «Поступление и выбытие»
ШАГ 7: ЗАПОЛНИТЕ СЛЕДУЮЩИЕ ПОЛЯ:
• «Врач ПО» – выбрать врача.
ДАЛЕЕ ПОЛЯ:
• «Время выбытия» – совпадает с датой приема на отделение.
• «Вид выбытия из ПО»;
• «Отделение» и «Профильное отделение»;
• «Профиль» - заполняем из справочника. (Открываем два раза левой кнопкой мыши)
• ДИАГНОЗ ПОСТУПЛЕНИЯ;



! Номер ИБ – присвоится автоматически;
ШАГ 8: По кнопке "Дальше" перейдите на вкладку «Протокол»: - выбираем из списка «Протокол поступления в ПО»;
Заполняем первичные данные по пациенту. Обязательны для заполнения поля, помеченные звездочкой (+ те, что заполняются автоматически, проверять);



ШАГ 9: Нажать кнопку далее и "Готово" .
Пациент поступил на отделение.
ШАГ 10: Печать согласий по кнопке «принтер» над списком пациентов


Алгоритм подписания протоколов (Эпикризы)

1. Перед подписанием любого эпикриза, должен быть проставлен диагноз поступления, вкладка «Диагнозы», в случаи выписного эпикриза и ДИАГНОЗ ВЫБЫТИЯ!





2. Добавляем нужный протокол в разделе текущая госпитализация – Дневники: заполняем все необходимые поля/данные - проверяем себя перед подписанием по кнопке "i".



3. В открывшемся окне, будет отображено, что не так, перед подписанием необходимо устранить.
• Пример: Такой тип ошибки означает, что у пациента не заполнены данные в шифрах/полисы;


Алгоритм создания протоколов ВК




1. Работаем в «Дневниках» - по кнопке добавить – выбираем тип записей «Эпикризы/ВК» нажимаем «Да»;



2. Вкладка «Заведение записей»
3. Нажимаем «+Новая запись»
4. Выбираем соответствующий эпикриз и выписку ВК в соответствии со случаем;
5. Подтверждаем действие по кнопке «Да»



6. Часть данных заполняется автоматически, пустые строчки заполняете вручную, обязательно проставляете дату протокола. Заполняем необходимые поля (период, данные по комиссии , номер ЭЛН и тд. - ориентир ЭЛН)
7. Доступны две формы печати : с ЭЦП и без ЭЦП;
8. Подписываем протокол по кнопке «ЭЦП»


Алгоритм работы врача в АРМ "Приемное отделение"




ШАГ 1: Войдите в систему АРМ «Приемное отделение»: введите свой пароль и нажмите кнопку «ОК»;
ШАГ 2: Выберите отделение: если вы работаете в нескольких отделениях, кликните один раз левой кнопкой мыши по нужному;
ШАГ 3: Подтвердите вход: нажмите кнопку «Да»;



ШАГ 4: Вкладка «Приемное отделение» откроется автоматически.
ШАГ 5: Проверьте фильтры: укажите нужную дату и отделение (по умолчанию система ставит текущий день);
Шаг 6: Найдите пациента:

 ВАРИАНТ 1: (через общий список): Выберите пациента в списке и откройте его медицинскую карту двойным кликом левой кнопкой мыши;




 ВАРИАНТ 2: (через поиск): Перейдите во вкладку «Поиск» - введите данные пациента (ФИО или СНИЛС) - нажмите кнопку «Найти» - затем в нижней части экрана выберите событие «Регистрация в ПО» и откройте его двойным кликом левой кнопкой мыши;

ДОБАВЛЕНИЕ ПРОТОКОЛА ОСМОТРА ВРАЧА ПРИЕМНОГО ОТДЕЛЕНИЯ



Шаг 7: откройте вкладку «Протоколы»;
Шаг 8: нажмите кнопку «+Добавить» и выберите из списка «Протокол врача приемного отделения»;
Шаг 9: Подтвердите действие: нажмите кнопку «Да»;
Шаг 10: Внесите информацию: заполните все необходимые поля данными;

ДОБАВЛЕНИЕ ЛАБОРАТОРНЫХ И ДИАГНОСТИЧЕСКИХ НАЗНАЧЕНИЙ ПАЦИЕНТУ
