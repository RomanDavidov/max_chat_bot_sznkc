🩺 Для Врачей и медицинских сестёр (Амбулатория ЦП)

📌 Содержание

    Инструкции для врачей и медицинских сестёр

    1. РАБОТА С НОВЫМИ ПРОТОКОЛАМИ ВК

    1.1. Находятся в папке Общие – ВК_ЦП



    1.2. В каждом врачебном протоколе (первичный, повторный) создан раздел ВК:



    1.3. Выбираем нужное, заполняем все необходимо в протоколе, далее подписываем ЭЦП, далее автоматически будет добавлен протокол - «Выписки из протокола подкомиссии по ЭВН».
    Осмотр подписывает врачом, Выписка подписывается ЭП членами комиссии и председателем.
    1.4. Эпикриз ЭВН (прошедший период), далее Выписка из протокола Прошедший период
    Эпикриз подписывается врачом, Выписка членами комиссии и председателем.
    1.5. Эпикриз на подкомиссию по ЭВН (Дубликат), далее Выписка из протокола подкомиссии по ЭВН (Дубликат)
    Эпикриз подписывается врачом, Выписка членами комиссии и председателем.
    1.6. Выписка из протокола ЭВН МСЭ
    • В каждом врачебном протоколе (первичный, повторный) создан раздел МСЭ



    Первая строчка для варианта когда мы направляем на ОБСЛЕДОВАНИЯ, необходимых для получения клинико-фукнциональных данных.
    Вторая строчка – когда уже выполнены исследования и пациента направлен на подкомиссию по ЭВН для решения вопроса о направлении на МСЭ.
    • При формировании Выписки из протокола созданы варианты ЭВН МСЭ1 и ЭВН МСЭ2
    ЭВН МСЭ 1 выбираем когда мы во врачебном протоколе выбрали первый вариант, т.е. для направления на обследования.
    • Если мы уже направляем для решения вопроса о направлении на МСЭ выбираем протокол «Выписка из протокола ЭВН МСЭ 2», где мы выбираем нужные значения.
    • Строчку с точкой мы пользуемся только в случае когда основания для МСЭ НЕ ВЫЯВЛЕНЫ.



    1.7 . Контроль заведующего (Для ЭЛН)
    Если в протоколе специалиста, заполнено поле Контроль заведующего.. заполняем все необходимо в протоколе, далее подписываем ЭЦП, далее автоматически будет добавлен протокол , автоматически будет добавлен протокол «Контроль заведующего (ЭВН)»
    Осмотр подписывается врачом, Контроль заведующего подписывается - ЭП председателем и заместителем.
